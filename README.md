# Comparacao da Inflacao entre Paises

Este projeto tem como objetivo analisar e comparar as taxas de inflacao de diferentes paises ao longo do tempo, utilizando dados economicos disponiveis publicamente.

## Descricao

A inflacao e um indicador economico crucial que reflete o aumento geral dos precos de bens e servicos em uma economia. Compreender as tendencias inflacionarias pode auxiliar na elaboracao de politicas economicas e na tomada de decisoes informadas por parte de empresas e individuos. Este projeto busca:

- Coletar dados historicos de inflacao de diversos paises.
- Limpar e preparar os dados para analise.
- Visualizar e comparar as taxas de inflacao entre os paises selecionados.

## Estrutura do Projeto

- `paises_europa/`: Contem dados especificos dos paises europeus utilizados na analise.
- `analises_infla.ipynb`: Notebook Jupyter com o codigo de analise e visualizacao das taxas de inflacao.

## Como Utilizar

1. **Clone o repositorio:**

   ```bash
   git clone https://github.com/GuilhermeFer29/Comparacao_inflacao_paises.git
   ```

2. **Instale as dependencias necessarias:**

   Certifique-se de que voce possui o [Python](https://www.python.org/) instalado. Em seguida, instale as bibliotecas requeridas:

   ```bash
   pip install -r requirements.txt
   ```

3. **Execute o notebook:**

   Abra o notebook `analises_infla.ipynb` utilizando o Jupyter Notebook ou Jupyter Lab para visualizar e interagir com as analises.
